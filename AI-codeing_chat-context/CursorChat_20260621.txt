画面崩れの原因を特定し、修正しました。元の画像は不要でした。.design/figma_ai と git 上の元ファイルを参照して復元しています。

原因
Tailwind CSS の設定が壊れていた
Next.js 移行時に index.css が Tailwind v3 記法（@tailwind base 等）に書き換えられ、v4 向けの @theme・bg-card・text-muted-foreground などのユーティリティが生成されていませんでした。

ページ構成が簡略化されていた
pages/ 配下がモック実装のままで、本来の TopPage・DetailPage 等が使われておらず、Layout（ナビ・サイドバー・フッター）も _app.tsx に組み込まれていませんでした。

修正内容
ファイル	内容
src/styles/index.css
元の Tailwind v4 記法に復元
pages/_app.tsx
Layout で全ページをラップ
pages/*.tsx
各 src/app/pages/* コンポーネントを使用
TopPage.tsx
未修正の navigate() を router.push() に修正
DetailPage.tsx / EditProfilePage.tsx
ルートパラメータの読み込みを修正
tailwind.config.cjs
v4 と競合するため削除
